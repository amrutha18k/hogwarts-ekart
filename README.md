# 🪄 Diagon Kart

A Harry Potter-inspired e-commerce web application built with **React** that allows users to browse magical products, search and filter items, view detailed product information, and manage a shopping cart.

🌐 **Live Demo:**  
https://diagon-kart.netlify.app/

---

## 📖 About

Diagon Kart is a frontend e-commerce application inspired by the magical world of Harry Potter. The project focuses on implementing real-world e-commerce functionality while showcasing modern React development practices, including reusable components, client-side routing, state management, and responsive UI design.

The application uses locally stored product data and was built as a portfolio project to strengthen frontend development skills.

---

## ✨ Features

- 🛍️ Browse magical products
- 🔍 Search products by name
- 🏷️ Filter products by category
- ⭐ Sort products by rating
- 📄 View detailed product information
- 🛒 Add and remove items from the shopping cart
- ➕ Update product quantities
- 💰 Automatic order summary and total price calculation
- 📱 Responsive design for desktop and mobile devices

## 🚀 Running Locally

```bash
# Clone the repository
git clone https://github.com/amrutha18k/diagon-kart.git

# Navigate into the project
cd diagon-kart

# Install dependencies
npm install

# Start the development server
npm run dev
```
